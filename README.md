  A Telegram Bot that works with FastAPI to communicate with WayForPay API and SQLAlchemy to store user data
</p>

This bot provides private chat and group accessible by subscription. Also, products which possible to buy in a bot and support chats with admins.
Admins can add, edit, or remove products.

To start the bot use:
```sh
python3 -n venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 main.py
```
